# Enterprise WG's (Sistema para Empresa)

Este é o frontend de um sistema gerencial focado em controle de estoque, registro de vendas (saída de estoque), cadastro de produtos e gerenciamento de permissões de usuários. Construído com HTML, CSS e JavaScript de forma limpa e otimizada.

## Teste Online

A aplicação está temporariamente configurada e hospedada para testes através do AWS S3. 

🔗 **[Acessar o Sistema Para Empresa (S3)](https://enterprise-wg-wanders.s3.us-east-1.amazonaws.com/index.html)**
